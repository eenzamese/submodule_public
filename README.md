## Installation

Clone the repo
   ```sh
   git clone https://github.com/eenzamese/submodule_public
   ```

## Usage

Edit inputs in speed_switch.py
```py
# inputs
APP_TMT = 60
SUCCESS_TMT = 600
FAIL_TMT = 60
ERR_TMT = 10
INET_HOST = '8.8.8.8'
```

Run python script
   ```sh
   python3 submodule_public/submodule_public.py
   ```
